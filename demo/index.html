Basic Snapper example

Just a snapper carousel without any external navigation. By default, item widths are calculated evenly based on the number of total items, so that one item fills 100% of the scroll pane at a time, and the scroller snaps to each item. Scrollbar is visible, which we recommend when not using additional navigation (like thumbnails or next/prev links).
